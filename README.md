# KOREAN 3B — 서울대 한국어+ (уроки 10–18)

PWA-приложение для изучения корейского. Чёрно-белый минималистичный дизайн (YZY),
7 вкладок, персональный словарь с сохранением и свайпом, два языка интерфейса (RU / EN).

## Вкладки
- **단어 (Слова)** — флешкарты по урокам  10–18 (190 слов). «Не знаю» → слово уходит в 사전.
- **문법 (Грамматика)** — 33 грамматики: простое объяснение + 2 диалога-примера с переводом.
- **듣기 (Аудирование)** — 10 диалогов муж/жен, вопросы (продолжение / что верно / главная мысль).
  Каждую реплику можно проиграть; если mp3 нет — включается голос браузера.
- **읽기 (Чтение)** — 10 текстов, **любое** корейское слово кликабельно и переводится
  (глоссарий текста + 190 слов урока + 182 базовых слова). Не нашли — «번역을 찾을 수 없어요».
  Кликнутые слова уходят в 사전. Плюс вопросы на понимание.
- **쓰기 (Письмо)** — 18 уровней (подразделы 10-1 … 18-2) по 10 заданий + режим «랜덤»
  (10 случайных из 180). Предложение с пропуском + 4 варианта. При ошибке показывается
  правильный ответ; в словарь ошибки НЕ попадают. Прогресс не сохраняется.
- **사전 (Словарь)** — только клики из 읽기 + «не знаю» из 단어/듣기.
  Свайп влево — удалить выученное слово. Сохраняется через window.storage.
- **정보 (Инфо)** — благодарность, просьба об обратной связи, Instagram @sawwaty.

## Как задеплоить на GitHub Pages
1. Создай репозиторий (например `korean-3b`) и загрузи ВСЕ файлы из этой папки,
   сохранив структуру (`index.html`, `app.js`, `data.js`, `manifest.json`, `sw.js`,
   папки `icons/` и `audio/`).
2. Settings → Pages → Branch: `main`, папка `/ (root)` → Save.
3. Через минуту приложение будет по адресу `https://<логин>.github.io/korean-3b/`.
4. Открой на телефоне → «Добавить на главный экран» — работает офлайн как приложение.

## Аудио (ElevenLabs)

Есть два способа. Самый простой — **прямо в приложении**:

### Способ 1 — кнопка в приложении (рекомендую)
1. Открой приложение → вкладка **정보 (Инфо)** → внизу нажми **⚙ Генерация аудио (ElevenLabs)**.
2. Вставь свой ключ ElevenLabs (профиль → API Keys). При желании впиши свои ID голосов
   (ElevenLabs → Voices → скопировать Voice ID); по умолчанию стоят женский и мужской.
3. Нажми **Сгенерировать и скачать**. Приложение обойдёт все 48 реплик, покажет прогресс
   и скачает `audio.zip`.
4. Распакуй `audio.zip` — внутри папка `audio/` с файлами `d1_0_F.mp3`, `d1_1_M.mp3` и т.д.
   Положи эту папку `audio/` в корень репозитория (рядом с `index.html`) и запушь.
   Ключ используется только для запроса к ElevenLabs и никуда больше не отправляется.

### Способ 2 — Python-скрипт (если удобнее из терминала)
1. `pip install requests`
2. Открой `generate_audio.py`, впиши свой `ELEVENLABS_API_KEY`.
3. `python3 generate_audio.py` — создаст mp3 прямо в папке `audio/`.

До появления mp3 приложение всё равно работает: реплики читает голос браузера.

## Файлы
- `index.html` — каркас + стили + регистрация service worker
- `app.js` — вся логика
- `data.js` — весь материал (слова, грамматика, тексты, диалоги, задания, UI-строки, базовая лексика)
- `data.json` — та же база в JSON (нужна только скрипту озвучки)
- `generate_audio.py` — генерация mp3 через ElevenLabs
- `manifest.json`, `sw.js`, `icons/` — PWA
