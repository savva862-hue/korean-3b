듣기 오디오 생성기 — Korean 3B

Генерирует озвучку всех диалогов 듣기 (уроки 10–18) через ElevenLabs прямо в браузере. Ключ отправляется только в официальный API ElevenLabs. Работает офлайн (ZIP собирается локально).

После скачивания:
1. Распакуй audio.zip — внутри папка audio/ с файлами вида d1_0_F.mp3.
2. Положи папку audio/ в репозиторий рядом с index.html (замени старую).
3. Commit + push. Через минуту звук заработает везде, включая iPhone.
Бесплатный лимит ElevenLabs — 10 000 символов/месяц. Этот набор из 48 реплик ≈ 2243 символов — с запасом.
